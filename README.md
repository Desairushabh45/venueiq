# VenueIQ

VenueIQ is a real-time smart companion web app for large-scale sporting venues that helps attendees navigate crowd movement, reduce waiting times, and stay informed through live coordination and emergency alerts.

## Problem Statement

Large-scale sporting venues often face challenges with crowd movement, resulting in congested areas and long waiting times for amenities like entry gates, food stalls, and restrooms. Lack of real-time coordination and communication can also lead to frustration or safety hazards during emergencies. VenueIQ addresses these issues by providing attendees and staff with live, actionable insights to optimize crowd flow, minimize wait times, and improve overall venue safety and experience.

## Features

- **Live Crowd Heatmap:** An interactive stadium map with color-coded zones (Green = Low, Yellow = Medium, Red = High) that displays real-time crowd congestion and density.
- **Queue Wait Time Tracker:** Live tracking of waiting times for essential venue amenities including entry gates, food and beverage stalls, and restrooms.
- **Real-Time Alert System:** A prominent dynamic notification system that delivers instant announcements, operational updates, and emergency alerts to ensure attendees stay informed.

## Tech Stack

- **Frontend:** React, Vite
- **Hosting:** Firebase Hosting
- **AI/Tooling:** Google Antigravity

## How to Run Locally

1. Clone the repository (if applicable) or navigate to the project directory:
   ```bash
   cd VenueIQ
   ```

2. Install the project dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` (or the port specified in your terminal).
